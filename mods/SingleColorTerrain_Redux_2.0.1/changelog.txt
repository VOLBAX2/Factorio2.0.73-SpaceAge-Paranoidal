Version: 2.0.0
Date: 2024-10-23
  Features:
    - Partial Space Age support is now included. Only nauvis will have terrain recolored at this time.
      - Terrain on the new planets seem to vary a lot more in regards to bonuses/uses so I'm not sure making them all look exactly the same is a good idea? Seems like it would affect knowing where you can plant gleba trees, at least.
      - The No decoratives option should work on all planets/space. You MAY need to disable the No Decoratives option if that causes issues on Gleba. I don't think decoratives are required but they are part of the tree definitions.
  Bugfixes:
    - Fixed that decoratives were removed from Nauvis even when the No Decoratives option was off.
